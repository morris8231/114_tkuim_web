# MVP Features List (Minimum Viable Product)

## 核心功能 (P0 - Must Have)

1.  **使用者系統 (User)**
    *   [ ] 簡單註冊/登入 (或單一使用者模式 for MVP demo)
    *   [ ] 記錄使用者基本資料 (暱稱)

2.  **章節與任務 (Content)**
    *   [ ] 預設種子資料：至少 3 個章節 (起步、構圖、光線)
    *   [ ] 任務瀏覽：查看任務詳情、教學文字

3.  **拍攝作業提交 (Core Loop)**
    *   [ ] 圖片上傳 (單張或多張)
    *   [ ] 填寫自我評量 (滑桿或 1-5 分)
    *   [ ] 寫下反思心得 (Textarea)
    *   [ ] 提交成功回饋

4.  **作品展示 (Gallery)**
    *   [ ] 顯示自己的所有作品
    *   [ ] 依時間排序

## 進階功能 (P1 - Nice to Have for Final)

1.  **解鎖機制 (Gamification)**
    *   [ ] 完成 N 個任務後解鎖下一章
    *   [ ] 進度條顯示

2.  **每月回顧 (Review)**
    *   [ ] 自動統計當月數據 (照片數、任務數)
    *   [ ] 成長曲線圖 (Chart.js 或純 CSS 長條圖)

3.  **篩選功能**
    *   [ ] 依照「題材」或「技能」篩選作品
