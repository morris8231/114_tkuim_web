# 水果挑選指南
## 組員 何睿純 李同岳

## 專案簡介

本專案是一個示範型的資訊展示平台，透過卡片式的版面與互動功能，介紹日常常見水果的挑選技巧。使用者可以瀏覽基本的選果原則、依類別篩選或搜尋特定水果，並點擊查看每一張卡片的詳細資訊。此外，網站包含聯絡表單，示範如何在前端實作 HTML5 驗證與自訂提示訊息。

## 使用技術

- **HTML5**：語意化標籤與清楚的內容分區。  
- **CSS3 / Bootstrap 5**：採用 Bootstrap 網格系統、預設元件與 RWD 響應式設計，搭配少量自訂 CSS 打造一致的配色與 hover 效果。  
- **JavaScript**：使用原生 DOM API (`querySelector`、`addEventListener`、`dataset`) 操作頁面元素，實作搜尋、篩選、折疊及表單驗證。  
- **Constraint Validation API**：表單驗證與自訂錯誤提示。  
- **Git 版控 / GitHub Pages**：建議將此專案上傳至 GitHub，並透過 GitHub Pages 部署公開展示。

## 功能特色

1. **選果技巧整理**：整合農業專家與食品安全建議，例如避免瘀傷【848165092777498†L120-L124】、輕壓感受熟度【848165092777498†L126-L131】、聞香辨甜度【848165092777498†L132-L135】、選擇鮮豔當季水果【848165092777498†L140-L151】等。
2. **卡片式水果介紹**：目前收錄蘋果、香蕉、莓果、柑橘、瓜類與梨子六種水果，每張卡片提供簡要挑選建議與延伸說明，內容引用自北達科他州立大學選果指南【486004043006604†L143-L154】【486004043006604†L200-L205】等。按下「更多資訊」即可展開細節。
3. **搜尋與篩選**：使用下拉選單與即時搜尋框，可依水果類別（仁果類、熱帶水果、莓果類、柑橘類、瓜類）或關鍵字快速篩選卡片。篩選邏輯完全在前端透過 JavaScript 操作 `dataset` 實現。
4. **表單驗證**：聯絡表單包含姓名、電子郵件、主旨與訊息欄位，採用 HTML5 `required` 及 `type="email"` 限制，並透過 Constraint Validation API 及 Bootstrap 樣式顯示錯誤訊息，送出成功後給予提示並重置表單。
5. **響應式設計**：利用 Bootstrap 的 `col-lg-4 col-md-6` 配置讓卡片在大螢幕三欄、中螢幕兩欄、小螢幕單欄排列，各區段上下間距透過 `py-5`、`mb-3` 調整。
6. **額外效果**：自訂 `.hoverable` 與 `.btn-raise` 類別，為卡片與按鈕加入輕微位移與陰影，提升整體互動感。頁尾動態顯示當前年份。

## 檔案結構

```
fruit_selection_project/
├─ index.html   # 主頁面，包含所有區段與 Bootstrap 語意結構
├─ style.css    # 自訂樣式，定義卡片 hover 與按鈕動畫等
├─ script.js    # 前端腳本，處理篩選、搜尋、折疊與表單驗證
└─ README.md    # 專案說明（即本檔案）
```

若需放置圖片，可建立 `assets/` 資料夾存放原圖，目前範例使用 [via.placeholder.com](https://via.placeholder.com) 佔位符。

## 使用方式

1. 下載或 clone 此專案至本地端。  
2. 直接以瀏覽器開啟 `index.html` 檔案，或使用如 Live Server 的擴充套件啟動本地伺服器。  
3. 如需部署到 GitHub Pages，請將整個專案上傳至 GitHub 儲存庫，於設定中的 Pages 功能選擇 `main` 分支作為來源。

## 來源與參考

本專案內容的水果挑選建議整理自以下資料：

- Fresh Farms 提供的選購指南中提到避免受損、聞香辨甜度、輕壓感受熟度、選擇顏色鮮豔並購買當季水果等建議【848165092777498†L120-L151】。
- 北達科他州立大學的《Focus on Whole Fruits: How to Select and Store Fruit》提供對各類水果的具體挑選與保存方法，例如蘋果應選擇外表光亮且重量較重的果實【486004043006604†L143-L147】；香蕉成熟時呈均勻黃色並帶有小斑點【486004043006604†L200-L205】；柑橘類需選擇有香氣且皮面光滑的果實【486004043006604†L235-L239】；瓜類應挑選沉重且表面呈霧面者【486004043006604†L290-L296】；梨子應無瘀傷且靠近蒂頭處輕壓有彈性【486004043006604†L328-L334】。

這些資訊只作為示範用，實際購買水果時仍以當地販售品質為準。
